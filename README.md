**Dashboard de Comunicação Interna - BELAS**

**Sobre a Empresa**
A BELAS é uma construtora e imobiliária fictícia utilizada como base para o desenvolvimento deste projeto acadêmico.

---

**Sobre o Projeto**
Este projeto consiste no desenvolvimento de um Dashboard de Comunicação Interna, com o objetivo de centralizar informações e melhorar a comunicação entre equipes administrativas e operacionais da empresa.

A aplicação é um protótipo funcional que consome dados de uma API REST externa, permitindo validar a interface e a integração cliente-servidor sem a necessidade de um backend próprio.

---

**Objetivo**
Melhorar a comunicação interna da empresa BELAS, promovendo maior alinhamento entre setores e redução de falhas operacionais.

---

**Público-Alvo**
- Engenheiros  
- Corretores  
- Equipe administrativa  
- Gestores  

---

**Funcionalidades**
- Listagem de funcionários (usuários)  
- Visualização de postagens  
- Visualização de comentários  

---

**API Utilizada**
Este projeto utiliza a API pública:

👉 https://jsonplaceholder.typicode.com/

**Mapeamento da API:**
- `/users` → funcionários  
- `/posts` → comunicados internos  
- `/comments` → interações  

---

**Arquitetura**
- Arquitetura Cliente-Servidor  
- Consumo de API REST  
- Organização baseada em MVC (Model-View-Controller)  

---

**Tecnologias**
- Python
- HTML  
- CSS  
- JavaScript  

---

**Limitações**
- Utiliza dados simulados (JSONPlaceholder)  
- Dependente da disponibilidade da API externa  
- Não possui autenticação de usuários  
- Não há persistência de dados
